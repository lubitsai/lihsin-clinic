# 07｜櫃檯與管理員操作手冊

後台網址：`https://booking.lhpedclinic.com.tw/admin`
閒置 30 分鐘自動登出（可於系統設定調整）；所有操作都有稽核紀錄。

## 一、每日流程（櫃檯）

1. **登入** → 首頁即為「今日預約總覽」：左欄時間、蔡醫師／李醫師各一欄（手機顯示卡片）。
2. 上方卡片顯示每位醫師「已預約／總名額／剩餘」。
3. 病人到診 → 點該筆的 **報到**；看診結束 → **完成**。
4. 時段過後仍未出現 → **未到**（會累計未到次數；誤按可用「撤銷未到」）。
5. 電話取消 → **取消**（輸入原因，系統自動通知病人）。
6. 要改時間 → **改期**（進入改期頁選新時段，原單自動標「已改期」）。
7. 日結：右上 **列印** 或 **匯出 CSV**（匯出檔證件號一律遮罩）。

### 常用篩選

上一天／今天／下一天、日期選擇器、早午晚診、醫師、門診類型、狀態，
搜尋框支援：姓名、電話、證件末碼、預約編號。

### 特別門診審核

「待確認」狀態（黃色⏳）為特別門診新單 → 確認後點 **確認**，病人會收到通知。

## 二、代客預約

「代客預約」頁 → 搜尋既有病人或建新病人 → 選門診/日期/時段（櫃檯可見全部時段）→ 建立。
若撞到「同日已有預約」「7 天 3 筆」「預約限制」，系統會要求輸入**覆寫理由**才能強制建立（留稽核）。

## 三、排班調整

「排班管理」三個分頁：

| 分頁 | 用途 |
|---|---|
| 固定週班表 | 每週固定班（星期×診別×醫師×時間×名額×是否開放線上） |
| 日期例外 | 全日休診／診別休診／醫師休診／代診／特殊時間／加診／封鎖時段／暫停門診類型 |
| 時段名額調整 | 單一時段手動加開名額（需原因） |

**重要**：例外若影響已存在的預約，系統會**先列出受影響病人**，不會刪除任何預約。
選擇：逐筆「改期」，或「批次診所取消＋通知」。兩者都會自動發 LINE／簡訊。

## 四、病人與未到管理

- 「病人管理」搜尋 → 病人頁可看：遮罩證件號、過去/未來預約、取消/未到次數、限制狀態、後台備註。
- 未到累計**超過 3 次**（第 4 次）系統自動限制其線上預約；前台只顯示「請致電診所」（不會出現「黑名單」字樣）。
- 解除／暫時解除（設到期日）／重設未到次數 → **僅管理員**，需輸入原因。
- 可能重複病歷（同名同生日等）列在病人管理頁下方，**僅管理員**可合併（輸入「合併」二字確認；原始資料保留）。
- 「顯示完整號碼」需管理員權限，且每次查看都寫入稽核。

## 五、管理員專屬

| 功能 | 位置 |
|---|---|
| 員工帳號建立/停用/重設密碼 | 員工帳號頁（櫃檯人員無法進入，也不能動任何帳號） |
| 兩步驟驗證（TOTP） | 員工帳號頁下方，掃 QR 後輸入 6 位數啟用 |
| 預約規則（開放天數/上限/未到門檻/截止時間） | 系統設定 |
| 門診類型（開放/審核/年齡/星期/診別/醫師/文案/顏色） | 系統設定 |
| 醫師新增/停用 | 系統設定（不限兩位） |
| 稽核紀錄查詢 | 稽核紀錄頁 |

## 六、醫師唯讀帳號（選用）

員工帳號頁建立角色「醫師唯讀」並對應醫師本人 → 該帳號登入只能看自己的當日預約，無任何操作按鈕。

## 七、常見問題

- **病人說收不到通知**：病人頁確認 LINE 綁定狀態；封鎖官方帳號會自動改發簡訊；可在總覽點「傳送通知」補發。
- **帳號被鎖**：連續 5 次密碼錯誤鎖 15 分鐘；急用時請管理員重設密碼。
- **誤標未到**：該筆點「撤銷未到」，次數自動回沖。
- **系統顯示「操作失敗」**：重新整理後再試；持續發生請聯絡維護人員並附上時間點（稽核紀錄可對照）。
