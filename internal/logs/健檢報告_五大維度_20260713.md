- **建議（均為技術層、零可見文字變更；屬 02 手冊決策，建議列待辦交院長裁示）**：
  - P1：字型 CSS 非同步化（preload＋swap 或字重精簡/子集化）——首頁 mobile 最大槓桿。

  - P3：hero-bg／og-image 產 webp 並以 `<picture>` 掛載。
