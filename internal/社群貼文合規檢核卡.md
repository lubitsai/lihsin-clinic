# 社群貼文合規檢核卡｜立欣診所（止血包 A）

> **版本：2026-07-11（初版，Claude Opus 4.8 撰）**
> **適用**：Facebook／Instagram／LINE 官方帳號／Google 商家檔案（GBP）貼文與回覆、實體文宣、任何**對外可見的行銷文字**。
> **不適用**：官網 HTML — 官網走 `internal/tools/validate_site.py`，**不重複**（本卡是給沒有驗證器把關的通路用的）。
> **權威順位**：合規事實以 `00_專案總覽索引.md` 第四節為準；本卡是其「社群版操作介面」。**第四節任何增修需院長逐字核可**，本卡隨之更新。
> **鐵律**：社群＝可見層＝最嚴格。官網可放進 keywords／llms 的 promotional 詞（「推薦」等），**在社群一律當可見區處理、禁用**。

## 一、30 秒發文前檢核（每則必過，照抄打勾）

發文前逐條確認，任一項打叉就退回改稿：

- [ ] **無自稱超級詞**：推薦／最佳／第一／首選／權威／資深／頂尖／唯一／最專業（第二節黑名單）
- [ ] **無促銷招徠語**：優惠實在／最便宜／保證／限時／名額有限／免費送（第二節黑名單）
- [ ] **無處方藥品名＋促銷**：處方藥（含 GLP-1 品牌名）不與價格／優惠／招徠語同框（§3-1、§3-2）
- [ ] **無療效保證**：根治／治癒／完全改善／無效退費／一針見效（醫療法禁誇大療效）
- [ ] **疫苗貼文四但書齊備**：仿單＋醫師評估＋現貨來電確認＋停打／延後但書（第四節模板）
- [ ] **無寫死評分數字**：不寫「5.0 星」「Google 滿分」；不自評、不以贈品換好評（§3-3）
- [ ] **價格用語中性**：可寫金額與品項，不加「實在／划算／最低」等招徠形容（§3-1）
- [ ] **未宣稱非具專科**：耳鼻喉／慢性病＝症狀處理＋轉介／控制非治癒；標明非急診（§3-5）
- [ ] **回覆評論未洩就醫事實**：不證實對方是病患、不提就診日期／病情／診療項目（第五節鐵律）
- [ ] **兒童肥胖用語**：不節食羞辱、用「健康體位」框架（§3-4）
- [ ] **無未核實資訊**：星等、看診時段、活動時間與現況一致（曾出現來源落差）
- [ ] **個資／肖像**：貼病人照片／對話截圖前，有無書面同意？（無＝不貼）

> **一句話原則**：社群貼文的角色是「衛教與資訊」，不是「招攬」。分不清時，把句子改成「陳述事實」而非「勸你來買」，多半就合規了。

## 二、禁用語黑名單（附合規替代寫法）

### 2-1 自稱超級詞（與驗證器 `FORBIDDEN_TERMS` 同步）

| 禁用 | 為什麼 | 改成 |
|---|---|---|
| 推薦（自稱）／最佳／首選／權威／頂尖／最專業 | 醫療法 §86 招徠性、誇大 | 陳述客觀資格：「健保特約」「兒童過敏氣喘訓練」「公衛碩士」 |
| 第一／唯一／資深 | 排他性自稱、難查證 | 刪除，或改客觀事實（「2024 年開業」「北區育德路」） |
| 少數（自誇語境） | 稀缺話術 | 刪除或改中性描述 |

> **例外（醫學事實／時序，非自稱，可用）**：第一線治療、第一代抗組織胺、第一次就診、最常見疾病之一。判準＝**在描述病症或流程，不是在誇診所**。

### 2-2 促銷招徠語（社群特別容易踩）

| 禁用 | 改成 |
|---|---|
| 優惠實在／划算／最便宜／破盤 | 只寫品項與金額本身，不加形容（例：「自費流感疫苗每劑 XXX 元」） |
| 保證／絕對／百分百／無效退費 | 「依醫師評估」「因人而異」 |
| 限時／最後幾天／名額有限／手刀搶 | 「預購至 X 月 X 日」「現貨有限，來電確認」（陳述事實，不製造搶購） |
| 免費送（贈品換行為） | 衛教資源可寫「免費索取」；**不得以贈品／折扣換 Google 評論或轉發** |

### 2-3 療效與藥品

| 禁用 | 為什麼 | 改成 |
|---|---|---|
| 根治／治癒／完全好／斷根 | 誇大療效（醫療法） | 「控制」「改善」「緩解」「長期追蹤」 |
| 一針見效／打了就不會生病 | 誇大＋不實 | 「降低感染與重症風險」「依仿單效力」 |
| 處方藥品名＋價格／優惠（含 GLP-1 品牌） | **藥事法處方藥廣告禁令** | 用泛稱＋off-label 說明，不與促銷同框（§3-2） |

## 三、情境專章（高頻踩雷場景）

### 3-1 價格貼文（疫苗預購、自費品項）
- 可寫：品項、劑量、金額、預購期限、預約方式。
- 不可寫：任何價格招徠形容詞（實在／划算／最低）、「比別家便宜」、限時搶購話術。
- 安全句型：「2026 自費流感疫苗開放預購至 X/X，共 N 款可選，數量有限，請來電或 LINE 確認現貨。」

### 3-2 減重／GLP-1 貼文
- **處方藥品名（猛健樂／Mounjaro／tirzepatide 等）不可與價格、優惠、招徠語同框**（藥事法）。
- 用泛稱：「腸泌素／GLP-1 類藥物」；如提及仿單外使用，標明「off-label、需醫師評估」。
- 兒童肥胖：不寫「快速瘦」「甩肉」；用「健康體位」「均衡飲食＋作息＋活動」三寶框架，不羞辱。

### 3-3 評論與評分
- **不自評、不寫死星數**（不寫「5.0 星滿分診所」）。
- **不得以抽獎／贈品／折扣換取或篩選好評**（違反 Google 政策＋醫療法）；抽獎若辦，須對**所有**評論者開放、不限定五星。
- 診後可中性邀請留評（「歡迎分享您的就診經驗」），不引導只留好評。

### 3-4 專科邊界
- 只宣稱具備的專科（兒童過敏氣喘、肥胖醫學、家醫）。
- 耳鼻喉相關症狀＝「評估與轉介」；慢性病＝「控制、追蹤」非「治癒」。
- 急症一律標「非急診，緊急請撥 119 或至急診」。

### 3-5 過敏檢測
- 「不單以檢測數值決定忌口」但書必附；不誇大檢測可「找出所有過敏原」。

## 四、疫苗貼文四但書（複製即用）

任何疫苗相關貼文，末段擇要帶入以下四點（依貼文長度可精簡，但四類不可全缺）：

```
※ 接種前請詳閱疫苗仿單；
※ 是否適合接種，須經醫師評估；
※ 現貨數量有限，請來電 06-2516086 或 LINE 確認；
※ 發燒、急性感染或特定情況需暫緩接種，請先諮詢醫師。
```

## 五、Google 評論回覆隱私鐵律（GBP 專用，最容易出事）

**回覆出自診所官方＝機構對外揭露就醫事實**，即使對方自己先寫出來，也**不得證實留言者是本院病患、不得提及任何就診日期／病情／診療項目**（個資法敏感範疇；負評尤其）。

- **好評模板**：「謝謝您的鼓勵與回饋，我們會持續照顧北區家庭的健康。——立欣診所」
- **負評模板**：「謝謝您的意見，我們很重視每一位民眾的感受。因涉及個人情況，不便於公開平台討論細節，誠摯邀請您來電 06-2516086 由專人了解與協助。」
- **禁止範例**：「您上週帶小朋友來打疫苗時等太久，我們很抱歉」← 一句話同時洩露就醫事實、日期與診療項目。
