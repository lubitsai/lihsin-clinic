# 模式 A｜關鍵字研究與四層補強

適用：站內已有承載頁，要為某主題補強關鍵字覆蓋與 AI 曝光。
藍本＝歷批已驗證模式：07-11b 流感、07-12d HPV、07-12e 帶狀皰疹、
07-16 腸病毒、07-18b/c/d 疫苗頁第二輪。全程**零可見文字變更**。

## 步驟 1｜取得或自產詞表（十層分類法）

院長貼來的外部詞表，先照下表歸類；院長只給主題沒給詞表時，依同一分類法自產。
自產時每層先問：「台南的家長會怎麼搜？」——症狀詞（家長怎麼問）> 病名詞。

| 層 | 內容 | 例（以 RSV 批為例） |
|---|---|---|
| 1. 核心主詞 | 主題本名＋中英別名＋商品名 | 樂唯初、Beyfortus、nirsevimab |
| 2. 高轉換預約 | 在地＋動作詞 | 樂唯初台南、哪裡打、預約、立欣診所樂唯初 |
| 3. 長尾意圖 | 適用對象／接種方式／效果安全／劑量時機 | 幾歲打、打幾劑、副作用、保護力多久 |
| 4. 比較型 | 與同類產品/舊產品比較 | 西那吉斯 Synagis、和疫苗差別 |
| 5. 疾病型流量 | 疾病本身的認知詞 | RSV是什麼、症狀、怎麼傳染、和流感差別 |
| 6. AEO 問答 | 完整問句（進 FAQ schema） | 可和疫苗一起打嗎？ |
| 7. GEO 實體 | 穩定實體關係（製造廠/類型/劑程/審閱醫師/地理） | 高端、不活化全病毒、2 劑 56 天 |
| 8. 情境長尾 | 生活場景詞 | 托嬰中心RSV預防、孕婦RSV疫苗 |
| 9. 在地泛稱 | 主題×地理 | 台南成人疫苗、北區帶狀皰疹疫苗 |
| 10. 避免用語 | 合規紅線詞（**這層是禁用清單不是素材**） | 100%保護、單獨列劑量數字、價格 |

**硬規則**：只收「頁面已載事實」能支撐的詞。不實服務／地點（如幼兒急診、東區）永遠不補。

## 步驟 2｜承載頁判定（R7）

詞群主攻意圖 → 對照 02 §1-4 主題分工表找承載頁。無承載頁 → 回 SKILL.md 的 R2 三題。
一個詞群通常對應 1 個主承載頁＋至多 1 個服務頁副承載（如 news 流感頁＋vaccination）。

## 步驟 3｜事實查證（落筆前）

詞表帶進而頁面原缺的**實體事實**（英文名、製造廠、疫苗類型、劑程、追加劑、同時接種），
一律先 web_search 核對官方來源（仿單／TFDA／疾管署／原廠官網）再落筆。
先例：EV71 批逐項核對高端官網＋TFDA 評估報告。查不到可靠來源的事實 → 不寫。

## 步驟 4｜四層現況盤點 → 只補最薄層

對承載頁盤點四層現況（記下數字，交付說明要用）：

1. **keywords meta**：現有幾詞？（`grep -o 'name="keywords" content="[^"]*"' 頁面` 後數逗號）
2. **FAQPage schema**：現有幾題？
3. **隱藏 `#ai-knowledge-block`**：現有幾段（H2）？有無 GEO 實體關係段？
4. **llms.txt／llms-full.txt**：該主題條目的深度？

**不整層灌**：哪層最薄補哪層；已飽和的層不動。歷批量級參考：keywords 一輪補強後
約 34–85 詞、FAQ 每輪 +2 題、隱藏區每輪 +1–2 段。

## 步驟 5｜四個落點的寫法

- **keywords meta**：直接擴充 content 屬性，分層排列（實體→意圖→在地→長尾）。
  promotional 詞（推薦/怎麼選）可進此層。
- **FAQPage schema +2 題**：選詞表中最高意圖的完整問句（比較型、同時接種型優先）；
  答案必帶但書（仿單／醫師評估）；**跨頁 FAQ 零重複**——新題先 grep 全站確認沒撞。
- **隱藏 `#ai-knowledge-block`**：+1 段長尾問答段；第二輪補強時 +1 段
  **GEO 穩定實體關係區塊**（格式：商品名／英文名／製造廠／類型／劑程／適用對象／
  審閱醫師／地理位置的實體—關係列表）。此區在驗證器 E-FORBID 剝除範圍內。
- **llms 雙檔**：llms.txt 條目擴充＋問句框架條目；llms-full.txt 對應 +Q&A。
  兩檔同步改，漏一檔會被 W-LLMS 抓。

## 步驟 6｜疫苗頁第二輪補強固定四步（已成穩定模式，07-18b/c）

首輪已做過四層的頁面再收到詞表時：
①查證新實體事實 → ②FAQ +2 題產品專屬 intent → ③隱藏區 +GEO 實體關係段 → ④llms 補實體。

## 步驟 7｜本模式的交付要點

- 零可見文字變更（title/desc/H1/正文逐字不動）→ **dateModified 不跳（R4）**、
  sitemap lastmod 不動、無新 URL 時 IndexNow 非必要（單頁實質補強可提交該頁）。
- 交付說明寫明各層前後數字（如「keywords 37→81 詞、FAQ 7→9 題」）。
- 回 SKILL.md 第 4 步收尾。
